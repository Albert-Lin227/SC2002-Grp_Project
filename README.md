# SC2002-Grp_Project
## Build & Run

1. Open a terminal and go to the project folder:
```bash
cd StudentMgr
```

2. Compile the sources:
```bash
javac -d bin src/h/*.java
```

4. Run the application:
```bash
java -cp bin h.MainApplication
```

5. Run the CLI Application:
```bash
java -cp bin h.CLIApplication
```
